# 计算器使用说明

> 本文档面向最终用户，说明如何启动和使用本 Web 计算器应用。

---

## 一、环境依赖与启动步骤

### 1.1 环境要求

- **操作系统**：Windows 10/11、macOS、Linux 均可
- **Python**：3.11 或 3.12
- **浏览器**：Chrome、Edge、Firefox、Safari 等现代浏览器（推荐 Chrome/Edge）
- **网络**：首次启动需要联网加载 Tesseract.js 模型（约 4MB）

### 1.2 启动步骤

#### 方式一：使用虚拟环境启动（推荐）

```bash

# 2. 使用项目虚拟环境的 Python 启动

# 或指定端口
F:\2023专业实习实训\.venv\Scripts\python.exe manage.py runserver 127.0.0.1:8080
```

#### 方式二：使用一键启动脚本

```bash
cd AHgroup\202321101246\任务三
F:\2023专业实习实训\.venv\Scripts\python.exe run.py
```

启动后控制台会显示：

```

[*] API 地址: http://127.0.0.1:8000/api/
```

### 1.3 浏览器访问

打开浏览器，访问以下地址：

```
http://127.0.0.1:8000/
```

或如果使用自定义端口（如 8080）：

```
http://127.0.0.1:8080/
```

> **注意**：如果浏览器提示 "0.0.0.0:8000 无法访问"，请使用 `127.0.0.1:8000` 或 `localhost:8000` 访问。

## 二、基础数值计算

### 2.1 界面说明

打开页面后，左侧为**计算器面板**，右侧为**公式识别面板**。

计算器面板包含：
- **模式选择栏**：可选择 19 种运算模式（默认"数值"）
- **输入框**：输入数学表达式
- **键盘区**：数字键、运算符、函数键
- **结果显示区**：显示计算结果
- **历史记录区**：显示最近计算记录

### 2.2 支持的函数与常量

#### 数学函数

| 函数 | 用法 | 示例 | 结果 |
|------|------|------|------|
| `sin` | `sin(x)` | `sin(pi/2)` | `1.0` |
| `cos` | `cos(x)` | `cos(0)` | `1.0` |
| `tan` | `tan(x)` | `tan(pi/4)` | `1.0` |
| `asin` | `asin(x)` | `asin(1)` | `1.570796...` (π/2) |
| `acos` | `acos(x)` | `acos(1)` | `0.0` |
| `atan` | `atan(x)` | `atan(1)` | `0.785398...` (π/4) |
| `log` | `log(x)` | `log(100)` | `2.0` (以10为底) |
| `ln` | `ln(x)` | `ln(e)` | `1.0` (自然对数) |
| `exp` | `exp(x)` | `exp(1)` | `2.718281...` (e) |
| `sqrt` | `sqrt(x)` | `sqrt(16)` | `4.0` |
| `pow` | `pow(a, b)` | `pow(2, 3)` | `8.0` |
| `abs` | `abs(x)` | `abs(-5)` | `5.0` |
| `floor` | `floor(x)` | `floor(3.7)` | `3.0` |
| `ceil` | `ceil(x)` | `ceil(3.2)` | `4.0` |
| `round` | `round(x)` | `round(3.5)` | `4.0` |
| `deg` | `deg(x)` | `deg(pi)` | `180.0` (弧度转角度) |
| `rad` | `rad(x)` | `rad(180)` | `3.141592...` (角度转弧度) |

#### 数学常量

| 常量 | 值 | 示例 |
|------|-----|------|
| `pi` | π ≈ 3.14159... | `sin(pi/2)` |
| `e` | e ≈ 2.71828... | `ln(e)` |

### 2.3 基本输入示例

```
1 + 2 * 3           → 7.0
(1 + 2) * 3         → 9.0
sin(pi/2) + cos(0)  → 2.0
sqrt(16) + log(100) → 6.0
pow(2, 10)          → 1024.0
```

### 2.4 键盘输入支持

| 按键 | 功能 |
|------|------|
| `Enter` | 执行计算 |
| `Escape` | 清空输入和结果 |
| `Backspace` | 删除最后一个字符 |
| 数字/字母 | 直接输入 |
| `+` `-` `*` `/` `^` | 运算符 |
| `(` `)` | 括号 |

## 三、高级运算模式

点击模式选择栏中的按钮，或在输入框中输入后按 **Enter** 切换模式。

### 3.1 数值计算（默认）

计算表达式的数值结果，使用 `=` 键执行。

```
表达式: 1+2*pow(3,2)
结果: 19.0
```

### 3.2 化简

化简代数表达式。

```
表达式: (x^2-1)/(x-1)
结果: x + 1
```

### 3.3 因式分解

将多项式因式分解。

```
表达式: x^2-1
结果: (x - 1)(x + 1)
```

### 3.4 展开

展开括号表达式。

```
表达式: (x+1)^2
结果: x**2 + 2*x + 1
```

### 3.5 方程求解

求解方程，输入格式为 `方程=0` 或 `方程=值`。

```
表达式: x^2-4=0
变量: x
结果: x = -2; x = 2
```

### 3.6 求导

对表达式求导。

```
表达式: x^3+2*x
变量: x
结果: 3*x**2 + 2
```

### 3.7 不定积分

计算不定积分，结果以显式积分符号展示。

```
表达式: x^2+1
变量: x
结果: ∫ x² + 1 dx = x³/3 + x + C
```

### 3.8 定积分

计算定积分，需输入**下限**和**上限**。

```
表达式: x^2
变量: x
下限: 0
上限: 1
结果: ∫₀¹ x² dx = 1/3
```

### 3.9 矩阵运算

支持矩阵加减乘，输入格式为 `[[a,b],[c,d]]`。

```
表达式: [[1,2],[3,4]]+[[5,6],[7,8]]
结果: [[6, 8], [10, 12]]
```

### 3.10 行列式

计算矩阵行列式。

```
表达式: [[1,2],[3,4]]
结果: -2
```

### 3.11 矩阵逆

计算矩阵的逆。

```
表达式: [[1,2],[3,4]]
结果: [[-2, 1], [1.5, -0.5]]
```

### 3.12 转置

矩阵转置。

```
表达式: [[1,2],[3,4]]
结果: [[1, 3], [2, 4]]
```

### 3.13 三角函数

三角函数化简与展开。

```
表达式: sin(x)^2+cos(x)^2
结果: 1
```

### 3.14 LaTeX

输出 LaTeX 格式。

```
表达式: (x^2+1)/(x-1)
结果: x^{2} + 1 / x - 1
```

### 3.15 极限（新增）

计算函数极限。

```
表达式: sin(x)/x
变量: x
结果: 1  (x→0 的极限)
```

### 3.16 泰勒展开（新增）

在 x=0 处泰勒展开。

```
表达式: sin(x)
变量: x
结果: x - x^3/6 + x^5/120 - ...
```

### 3.17 级数（新增）

级数求和。

```
表达式: 1/n^2
变量: n
结果: pi^2/6  (n 从 1 到 ∞)
```

### 3.18 特征值（新增）

计算矩阵特征值。

```
表达式: [[1,2],[3,4]]
结果: {5/2 - sqrt(33)/2: 1, 5/2 + sqrt(33)/2: 1}
```

### 3.19 矩阵秩（新增）

计算矩阵的秩。

```
表达式: [[1,2],[3,4]]
结果: 2
```

## 四、图片公式识别（OCR）

### 4.1 使用方法

1. **上传图片**：点击右侧"公式识别"面板的拖拽区域，选择图片文件
2. **拖拽上传**：直接将图片文件拖拽到上传区域
3. **粘贴截图**：截图后（如 QQ/微信截图），在网页任意位置按 `Ctrl+V`，图片自动粘贴并触发识别

### 4.2 识别结果处理

- 识别成功后，结果显示在"识别结果"区域
- 点击"应用到计算器"按钮，自动将识别结果填入左侧计算器输入框
- 识别记录保存在浏览器本地，刷新页面不会丢失

### 4.3 注意事项

- 图片尽量清晰，避免过度模糊
- 公式图片建议只包含公式本身，避免周围文字干扰
- 首次识别会加载 Tesseract.js 模型（约 4MB），请耐心等待
- 对简单公式（如 `x^2+1`、`sin(x)`）识别率较高，复杂分数和根号可能不够准确

## 五、sympy 未安装时的提示

如果后端 sympy 未安装，高级模式（化简、积分、矩阵等）会显示：

```
高级符号运算需要安装 sympy（当前环境未安装）
```

此时：
- **数值计算**仍然可用（基础计算器不受影响）
- **积分显式渲染**不可用
- 建议按照 AI交互说明.md 中的步骤安装 sympy

## 六、常见问题解答

### Q1: 浏览器提示 "0.0.0.0:8000 无法访问"

**A**：使用 `127.0.0.1:8000` 或 `localhost:8000` 访问，不要用 `0.0.0.0`。

### Q2: 页面报错 "ModuleNotFoundError: No module named 'sympy'"

**A**：使用项目虚拟环境的 Python 启动，不要用系统默认 Python：

```bash
F:\2023专业实习实训\.venv\Scripts\python.exe manage.py runserver
```

### Q3: 端口被占用，启动失败

**A**：关闭所有 Python 进程后重新启动：

```bash
taskkill /F /IM python.exe
python manage.py runserver
```

### Q4: 积分结果显示为纯文本而不是数学公式

**A**：检查网络是否能访问 MathJax CDN。如果无法联网，积分结果将以 LaTeX 纯文本显示，但不影响计算正确性。

### Q5: 图片识别结果乱码或错误

**A**：
1. 确保截图清晰，只包含公式
2. 尝试重新截图，避免背景过于复杂
3. 简单公式（如 `x^2+1`）识别率较高，复杂分数可能识别失败
4. 可手动修正识别结果后点击"应用到计算器"

### Q6: 如何清空计算历史？

**A**：点击历史记录区域的"清空历史"按钮。注意：计算历史保存在内存中，页面刷新后会重置。

### Q7: 模式按钮太多，显示不全

**A**：点击模式栏右侧的 ▾ 按钮，展开显示所有 19 个功能按钮。
